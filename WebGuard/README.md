# WebGuard - أداة الفحص الأمني
![صورة توضيحية](https://via.placeholder.com/800x400)

أداة متكاملة لاكتشاف الثغرات الأمنية في المواقع مثل:
- SQL Injection
- مشاكل Forms
- أخطاء SSL

## كيفية الاستخدام
1. نزل الملفات
2. شغل `chmod +x install.sh`
3. نفذ `./install.sh`
4. عدل ملف `websites.txt`
5. شغل `python3 scanner.py`
